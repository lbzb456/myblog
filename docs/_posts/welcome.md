欢迎来到 **行者小二** 的技术博客！这是一个使用 VuePress 框架构建的现代化博客网站。

## 博客特性

- **响应式设计**：完美适配各种设备
- **代码高亮**：支持多种编程语言
- **Markdown 支持**：使用熟悉的 Markdown 语法
- **分类和标签**：轻松组织和查找文章

## 技术栈

本博客使用以下技术构建：

- **VuePress**：Vue 驱动的静态网站生成器
- **GitHub Pages**：免费托管服务

## 开始使用

如果你也想创建自己的技术博客，可以参考以下步骤：

1. 安装 Node.js
2. 安装 VuePress
3. 初始化博客项目
4. 编写文章并部署
